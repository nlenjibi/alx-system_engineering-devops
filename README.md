Directory: `0x03-shell_variables_expansions`
All scripts: Bash, exactly 2 lines, executable

Scripts

- `3-paths`: counts directories in `PATH`.
- `4-global_variables`: lists environment variables (`printenv`).
- `5-local_variables`: lists locals, env, functions (`set`).
- `6-create_local_variable`: creates local `BEST=School`.
- `7-create_global_variable`: exports global `BEST=School`.
- `8-true_knowledge`: prints `TRUEKNOWLEDGE + 128`.
- `9-divide_and_rule`: prints `POWER / DIVIDE`.
- `10-love_exponent_breath`: prints `BREATH ** LOVE`.
- `11-binary_to_decimal`: base2 in `$BINARY` to decimal.
- `12-combinations`: all aa..zz combos except `oo`, one per line.
- `13-print_float`: prints `$NUM` with two decimals.
- `100-decimal_to_hexadecimal`: decimal `$DECIMAL` to hex.
- `101-rot13`: ROT13 encode/decode ASCII.
